# Where to Find Delete and Download Buttons

## Organizer Dashboard - Tab Navigation

The delete and download buttons are located in different tabs:

### 📸 Capture Tab
**Delete Buttons:**
- **Snapshot Delete**: 🗑️ button next to each snapshot in history
- **Capture Delete**: 🗑️ button next to each capture request (deletes images too)

**Location**: Click "Capture" tab → Scroll to see Snapshot Manager and Capture History

---

### 📝 Quiz Tab
**Download Button:**
- **📥 Download Report**: Top-right of Quiz Management section
- Downloads CSV with questions, answers, and attendee responses

**Delete Button:**
- **🗑️ Delete Question**: Next to each question (when expanded)

**Location**: Click "Quiz" tab → Scroll down to "Quiz Questions & Responses"

---

### 👥 Monitor Tab
**Delete Buttons:**
- **🗑️ Delete Attendee**: In "Actions" column of attendance table
- **🗑️ Clear All**: Top-right of attendance section (bulk delete)

**Location**: Click "Monitor" tab → Scroll to "Attendee Attendance" section

---

## Summary

| Feature | Tab | Location | Button |
|---------|-----|----------|--------|
| Delete Snapshot | Capture | Snapshot Manager | 🗑️ |
| Delete Capture | Capture | Capture History | 🗑️ |
| Delete Attendee | Monitor | Attendance Table | 🗑️ |
| Bulk Delete Attendance | Monitor | Attendance Header | 🗑️ Clear All |
| Download Quiz Report | Quiz | Quiz Management | 📥 Download Report |
| Delete Question | Quiz | Quiz Management | 🗑️ (when expanded) |

---

## Navigation

1. Go to Organizer Dashboard
2. Look for tabs at the top: **Monitor | Chains | Capture | Quiz | Session**
3. Click the relevant tab
4. Scroll to find the section with delete/download buttons

---

**Note**: All delete operations require confirmation and are logged for audit purposes.
